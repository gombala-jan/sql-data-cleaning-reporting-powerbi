# sql-data-cleaning-reporting-powerbi
SQL Data Cleaning und Reporting Pipeline mit MySQL und Power BI (Superstore Dataset)

Projektbeschreibung

In diesem Projekt wurde ein fehlerhafter Sales-Datensatz mit MySQL systematisch bereinigt und anschließend für Reporting- und Analysezwecke strukturiert aufbereitet.

Ziel war es, inkonsistente Rohdaten in eine saubere Reporting-Tabelle zu transformieren und daraus ein interaktives Management-Dashboard in Power BI zu erstellen.

Verwendete Tools
MySQL
SQL
Power BI
Durchgeführte Data Cleaning Schritte

Im Rahmen des Projekts wurden folgende Bereinigungsschritte umgesetzt:

Standardisierung der Spaltennamen
Konvertierung der Datumsformate
Identifikation und Behandlung fehlender Werte
Entfernung leerer Textfelder
Identifikation und Entfernung doppelter Datensätze
Validierung numerischer Werte (Sales)
Prüfung der Datumslogik
Erstellung einer finalen Reporting-Tabelle
Reporting & Dashboard

Nach der Datenbereinigung wurde eine strukturierte Reporting-Tabelle erstellt und in Power BI importiert.

Auf dieser Grundlage wurde ein interaktives Dashboard entwickelt zur Analyse von:

Gesamtumsatz
Anzahl der Bestellungen
Kundenanzahl
Umsatz nach Kategorie
Umsatzentwicklung im Zeitverlauf
Umsatz nach Region
Versandarten
Top-Produkten nach Umsatz
Zentrale Erkenntnisse aus der Analyse

Die Analyse zeigt:

Kategorie Furniture generiert den größten Umsatzanteil
Versandart Standard Class wird am häufigsten verwendet
Region West erzielt den höchsten Gesamtumsatz
Das Produkt Riverside Palais Royal Lawyers Bookcase ist der Top-Performer nach Umsatz
